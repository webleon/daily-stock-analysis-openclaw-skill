## 📱 Telegram 发送说明

### 配置要求

需要在 `.env` 文件中配置：

```bash
TELEGRAM_BOT_TOKEN=your_bot_token_here
TELEGRAM_CHAT_ID=your_chat_id_here
```

### 发送方法

```bash
# 激活虚拟环境
source venv/bin/activate

# 运行发送脚本
python scripts/send_to_telegram.py
```
