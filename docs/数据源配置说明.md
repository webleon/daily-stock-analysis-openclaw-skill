## 📝 PE 数据缺失原因

**AkShare 接口问题：**
- 列名：`市盈率 - 动态` 存在
- 数据：全市场所有股票 PE 都是 `N/A`
- 原因：接口未返回有效数据

**验证结果：**
```python
import akshare as ak
df = ak.stock_zh_a_spot_em()
print(df['市盈率 - 动态'].isna().sum())  # 输出：58（全部缺失）
```

---

## 🎯 解决方案

### 方案 1：配置 Tushare Pro（推荐 ⭐⭐⭐⭐⭐）

**步骤：**

1. **注册 Tushare**
   - 访问：https://tushare.pro/
   - 注册账号

2. **获取积分**
   - 新用户注册送 100 积分
   - 需要 120 积分才能获取 PE 数据
   - 获取积分方式：
     - 充值（1 元 = 1 积分）
     - 完成任务
     - 推广

3. **获取 Token**
   - 登录后访问：https://tushare.pro/user/token
   - 复制 Token

4. **配置环境变量**
   ```bash
   # 编辑 .env 文件
   vim ~/.openclaw/workspace/skills/daily-stock-analysis-openclaw-skill/.env
   
   # 添加配置
   STOCK_DATA_SOURCE=tushare
   TUSHARE_TOKEN=你的 token
   ```

5. **验证配置**
   ```bash
   cd ~/.openclaw/workspace/skills/daily-stock-analysis-openclaw-skill
   source venv/bin/activate
   python3 scripts/generate_comparison_html.py
   ```

**效果：**
- ✅ PE 数据准确显示
- ✅ 自动切换数据源
- ✅ 无需修改代码

## ⚠️ 注意事项

1. **积分要求**
   - Tushare 需要 120 积分才能获取 PE 数据
   - 新用户注册送 100 积分
   - 需要额外获取 20 积分

2. **调用限制**
   - Tushare 免费用户每日调用次数有限
   - 建议配置缓存

3. **数据更新**
   - AkShare：实时更新（但 PE 缺失）
   - Tushare：盘后更新（准确）

---

## 🎯 推荐配置

**最佳实践：**
1. 平时使用 AkShare（免费，数据基本够用）
2. 需要 PE 数据时切换到 Tushare
3. 或直接配置 Tushare（如果有积分）

**配置示例：**
```bash
# 主数据源：Tushare（准确）
STOCK_DATA_SOURCE=tushare
TUSHARE_TOKEN=your_token
```
