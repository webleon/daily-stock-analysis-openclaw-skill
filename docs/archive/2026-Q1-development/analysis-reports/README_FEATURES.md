# OpenClaw Daily Stock Analysis - 功能文档

## 📋 功能概览

本 Skill 提供完整的股票分析功能，包括单股分析、批量分析、大盘复盘、以及 11 种专业分析策略。

---

## 🚀 核心功能

### 1️⃣ **单股分析** (`/ask`)

使用特定策略分析单只股票。

**用法：**
```bash
/ask 600519                        # 使用默认策略（多头趋势）
/ask 600519 用缠论分析              # 自动识别策略
/ask 600519 chan_theory            # 直接指定策略 ID
/ask hk00700 波浪理论看看           # 港股分析
```

**支持策略：**
- `bull_trend` - 多头趋势（默认）
- `ma_golden_cross` - 均线金叉
- `volume_breakout` - 放量突破
- `shrink_pullback` - 缩量回踩
- `chan_theory` - 缠论分析
- `wave_theory` - 波浪理论
- `box_oscillation` - 箱体震荡
- `emotion_cycle` - 情绪周期
- `bottom_volume` - 地量见底
- `dragon_head` - 龙头战法
- `one_yang_three_yin` - 一阳穿三阴

**输出示例：**
```markdown
## 趋势判断
当前市场处于上升趋势，MA5/MA10/MA20 呈多头排列...

## 量能分析
今日成交量 2954 手，量比 0.88，属于正常水平...

## 操作建议
建议以 MA5（344 元）为防守位继续持有...
```

### 2️⃣ **批量分析** (`/batch`)

批量分析自选股列表中的所有股票。

**用法：**
```bash
/batch      # 分析所有自选股
/batch 3    # 只分析前 3 只
```

**输出示例：**
```markdown
✅ **批量分析任务已启动**

• 分析数量：10 只
• 股票列表：600519, 000858, 002594...

分析完成后将自动推送汇总报告。
```

**汇总报告：**
```markdown

✅ 分析完成：10 只
❌ 分析失败：0 只

## 1. 600519
当前市场处于上升趋势，建议以 MA5 为防守位持有...

## 2. 000858
...
```

## 📊 11 种分析策略详解

### 1. **多头趋势** (`bull_trend`)

**核心理念：** 先看指数方向，再看量能结构，最后看板块持续性。

**适用场景：** 趋势跟踪，适合大多数情况。

**分析维度：**
- 趋势结构：判断市场处于上升/震荡/防守阶段
- 资金情绪：识别短线风险偏好与情绪温度
- 主线板块：提炼可交易主线与规避方向

**操作框架：**
- 进攻：指数共振上行 + 成交额放大 + 主线强化
- 均衡：指数分化或缩量震荡，控制仓位并等待确认
- 防守：指数转弱 + 领跌扩散，优先风控与减仓

---

### 2. **均线金叉** (`ma_golden_cross`)

**核心理念：** 金叉买入，死叉卖出。

**适用场景：** 趋势明确的市场。

**分析维度：**
- 均线形态：MA5/MA10/MA20 的排列状态
- 成交量确认：验证金叉信号的有效性
- 风险控制：设置止损和止盈位

**操作框架：**
- 买入：MA5 上穿 MA10 + 放量确认
- 持有：均线多头排列 + 量价配合
- 卖出：MA5 下穿 MA10 或跌破止损位

---

### 3. **放量突破** (`volume_breakout`)

**核心理念：** 放量突破关键阻力位是买入信号。

**适用场景：** 突破行情，捕捉主升浪。

**分析维度：**
- 突破形态：是否突破关键阻力位
- 成交量分析：量比是否大于 2
- 持续性判断：板块是否配合

**操作框架：**
- 买入：放量突破关键阻力位 + 板块配合
- 加仓：缩量回调至支撑位
- 止损：跌破突破起点或支撑位

---

### 4. **缩量回踩** (`shrink_pullback`)

**核心理念：** 上升趋势中的缩量回踩是低吸机会。

**适用场景：** 上升趋势中的回调。

**分析维度：**
- 趋势判断：是否处于上升趋势
- 回踩分析：回踩幅度和量能
- 买入时机：是否在支撑位企稳

**操作框架：**
- 低吸：上升趋势中的缩量回踩支撑位
- 加仓：再次放量上涨确认
- 止损：跌破支撑位止损

---

### 5. **缠论分析** (`chan_theory`)

**核心理念：** 通过笔、段、中枢判断买卖点。

**适用场景：** 精确捕捉转折点。

**分析维度：**
- 笔段分析：识别当前的笔和段
- 中枢分析：判断中枢的位置和状态
- 背驰判断：判断是否形成背驰

**操作框架：**
- 一类买点：趋势背驰后的反转
- 二类买点：回踩不破前低
- 三类买点：突破中枢后回踩

---

### 6. **波浪理论** (`wave_theory`)

**核心理念：** 5 浪上升，3 浪下跌。

**适用场景：** 周期性明显的行情。

**分析维度：**
- 浪型判断：当前处于什么浪型
- 浪型验证：成交量、时间周期、幅度
- 目标位预测：使用黄金分割预测

**操作框架：**
- 买入：确认第 3 浪启动
- 持有：第 3 浪和第 5 浪持有
- 卖出：第 5 浪结束或 ABC 调整开始

---

### 7. **箱体震荡** (`box_oscillation`)

**核心理念：** 箱体下沿买入，上沿卖出。

**适用场景：** 震荡市。

**分析维度：**
- 箱体识别：识别箱体的上下沿
- 位置判断：当前价格在箱体中的位置
- 突破判断：是否形成突破

**操作框架：**
- 低吸：接近箱体下沿 + 缩量企稳
- 高抛：接近箱体上沿 + 放量受阻
- 突破：放量突破箱体后追涨

---

### 8. **情绪周期** (`emotion_cycle`)

**核心理念：** 情绪周期分为冰点、回暖、高潮、退潮。

**适用场景：** 短线操作。

**分析维度：**
- 情绪判断：涨停家数、连板高度、跌停家数
- 龙头分析：龙头股的状态
- 周期判断：情绪周期的阶段

**操作框架：**
- 买入：情绪冰点 + 龙头股企稳
- 持有：情绪回暖 + 龙头股连板
- 卖出：情绪高潮 + 龙头股见顶

---

### 9. **地量见底** (`bottom_volume`)

**核心理念：** 地量见地价，天量见天价。

**适用场景：** 左侧交易，捕捉底部。

**分析维度：**
- 地量判断：成交量是否创近期新低
- 底部特征：是否止跌企稳
- 确认信号：是否放量上涨

**操作框架：**
- 观察：地量出现后等待确认
- 试仓：出现确认信号后小仓位试仓
- 加仓：放量上涨确认后加仓

---

### 10. **龙头战法** (`dragon_head`)

**核心理念：** 只做龙头，不做跟风。

**适用场景：** 强势股操作。

**分析维度：**
- 龙头识别：连板高度最高、成交量最大
- 龙头状态：是否继续连板
- 参与时机：龙头首阴或回踩均线

**操作框架：**
- 买入：龙头首阴或回踩均线
- 持有：龙头继续连板持有
- 卖出：龙头见顶信号出现

---

### 11. **一阳穿三阴** (`one_yang_three_yin`)

**核心理念：** 一阳穿三阴是强烈的反转信号。

**适用场景：** 底部反转。

**分析维度：**
- 形态识别：前面是否连续三根阴线
- 量能分析：阳线是否放量
- 后续确认：次日是否继续放量

**操作框架：**
- 买入：一阳穿三阴 + 放量确认
- 加仓：次日继续放量上涨
- 止损：跌破阳线实体底部

## 🔧 配置说明

### 环境变量

```bash
# 股票列表（逗号分隔）
STOCK_LIST=600519,000858,002594

# LLM 配置
GEMINI_API_KEY=your_key
ANTHROPIC_API_KEY=your_key
OPENAI_API_KEY=your_key
```

### 策略选择

在 `/ask` 命令中可以使用中文或英文指定策略：

```bash
/ask 600519 缠论         # 中文
/ask 600519 chan_theory  # 英文
```

**策略名称映射：**
- 缠论/缠论分析 → `chan_theory`
- 波浪/波浪理论 → `wave_theory`
- 箱体/箱体震荡 → `box_oscillation`
- 情绪/情绪周期 → `emotion_cycle`
- 趋势/多头趋势 → `bull_trend`
- 均线金叉/金叉 → `ma_golden_cross`
- 缩量回踩/回踩 → `shrink_pullback`
- 放量突破/突破 → `volume_breakout`
- 地量见底 → `bottom_volume`
- 龙头/龙头战法 → `dragon_head`
- 一阳穿三阴 → `one_yang_three_yin`

## 📊 输出格式

所有分析报告都使用 Markdown 格式，包含以下部分：

1. **趋势/形态判断** - 明确判断当前状态
2. **数据分析** - 基于数据详细分析
3. **操作建议** - 具体的买入/持有/卖出建议
4. **风险提示** - 需要关注的风险点
