## ⚠️ 注意事项

### 1. 文件冲突

**最高风险**:
- `src/analyzer.py` - 你有 OpenClaw 多 Agent 定制
- `src/core/pipeline.py` - OpenClaw 可能有定制

**策略**:
```bash
# 使用 mergetool
git mergetool

# 或手动合并
git checkout --ours <file>  # 保留我们的版本
git checkout --theirs <file>  # 保留他们的版本
```

### 2. 依赖更新

检查 `requirements.txt`:
```bash
# 对比
diff <(git show upstream/main:requirements.txt) requirements.txt

# 合并新依赖
```

### 3. 数据库迁移

如果有数据库 schema 变更：
```bash
# 检查是否有迁移脚本
ls scripts/migrations/

# 运行迁移
python scripts/migrations/<migration>.py
```
